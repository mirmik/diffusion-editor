## Проект
Программа для рисования с нейросетками, с референсом в виде гимпа. 
За скоростью разработки не гонимся, нам важна архитектурная зрелость. 
Изначально разрабатывался на qt6, потом портирован на termin-gui.

## Язык
Предпочтительный язык общения - Русский.

## Ui ядро
Проект в качестве gui использует самописное ядро, которое в процессе работы можно и нужно дорабатывать.
Используются проекты:
/home/mirmik/project/termin-env/termin-base
/home/mirmik/project/termin-env/termin-graphics
/home/mirmik/project/termin-env/termin-gui
Эти библиотеки вынесены из 3д движка /home/mirmik/project/termin-env/termin в самостоятельные проекты.
